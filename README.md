### 1. Real-Time Task Scheduler
**Description:**  
A simulation of a CPU task scheduler using **FCFS**, **Round Robin**, and **Priority Scheduling** algorithms.

**Key Features:**
- Implements multiple scheduling strategies.
- Calculates average waiting and turnaround times.
- Demonstrates queue and heap-based scheduling.

**Concepts Used:** Queues, Min-Heaps, Sorting, Algorithm Analysis  
**File:** `real_time_task_scheduler.cpp`

---

### 2. Graph Traversal Visualizer
**Description:**  
A console-based visualizer for graph traversal using **Breadth-First Search (BFS)** and **Depth-First Search (DFS)**.

**Key Features:**
- Supports both directed and undirected graphs.
- Allows user-defined input for nodes and edges.
- Displays real-time traversal order.

**Concepts Used:** Graphs, Queues, Stacks, Recursion  
**File:** `graph_traversal_visualizer.cpp`

---

### 3. Expression Tree Evaluator
**Description:**  
A program that constructs and evaluates arithmetic expressions using a **Binary Expression Tree**.

**Key Features:**
- Builds trees from postfix expressions.
- Evaluates expressions recursively.
- Supports infix, prefix, and postfix traversals.

**Concepts Used:** Trees, Stacks, Recursion, Expression Parsing  
**File:** `expression_tree.cpp`

---

### 4. Dynamic Memory Allocator (Custom Heap Manager)
**Description:**  
A simulation of dynamic memory allocation using a **First Fit** strategy.

**Key Features:**
- Allocates and deallocates memory blocks interactively.
- Merges adjacent free blocks automatically.
- Displays real-time memory layout.

**Concepts Used:** Vectors, Structs, Memory Simulation, Algorithm Design  
**File:** `memory_allocator.cpp`
